CSS 教學

在這堂課程上面，你了解如何使用CSS設定使用在HTML標籤上，以及設定連結顏色。

這章節你認識的CSS設定文字語法

  1. 文字的顏色:color
  2. 文字的大小:font-size
  3. 文字的行距:line-height
  4. 文字的字體:font-family
  5. 其它...

下面的ement語法，哪一項是錯誤的?

  • ! = HTML5的環境
  • link:css = 匯入CSS
  • p*4 = 顯示4個p段落
  • h1 = 展開h1的開頭與結尾html標籤
  • lorem = 自動就幫我們寫完網頁
